# Blog

# Function List

**Backend**

1. ***books***
   * Paginate Books
     * type : Graphql Query 
2. ***createBook***
   * create a book
     * type: Graphql Mutation
3. ***deleteBook***
   - Delete a book
     * type: Graphql Mutation 

**Frontend**

1. ***/***
   * Home
2. ***/books***
   * List books, searching, paging 
3. ***/books/new***
   * create a book 
4. ***/users***
   * list users
5. ***/users/[username]***
   * display a user

# Infrastruction

**Backend**

* typescript
* apollo-server
* prisma + nexus

**Frontend**

* typescript
* next
* react
* final-form, react-final-form
* use-debounce
* tailwindcss
